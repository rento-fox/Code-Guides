How to Run Files:

    In the VS Code terminal, navigate to the directory where your files are located.
    Then run the correct command for each file:

        Python:
            python '.\filename.py'

        JavaScript:
            node '.\filename.js'

        Java:
            java '.\filename.java'

        C++:
            First compile the program:
                g++ -o Cpp filename.cpp
            This will create a "Cpp.exe" file. Run it with command:
                .\Cpp.exe

<br/>
